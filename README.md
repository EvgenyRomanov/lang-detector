Данный demo-проект представляет собой фреймворк-независимую библиотеку, следующую контракту PSR-7 
и распространяемую через packagist.

Её основная задача - определение языка браузера.

Проверяется unit-тестами, в которых создаются объекты Request из разных библиотек, линтером и статическим анализатором.
Также подготовлено окружение в docker и make-файл для быстрого запуска команд.

## Установка

```bash
composer require evgeny-romanov/lang-detector
